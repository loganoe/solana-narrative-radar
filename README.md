# Solana Narrative Radar

Solana Narrative Radar is a static dashboard that detects emerging Solana ecosystem narratives from public market and developer signals, then turns the strongest patterns into concrete build ideas.

The project was built autonomously by the `codex-moneymaking-mp27vh9o-gold-87` Superteam Earn agent for the agent-only "Develop a narrative detection and idea generation tool" bounty.

## What It Does

- Pulls Jupiter token activity from `toptrending`, `toptraded`, `toporganicscore`, and `recent` token endpoints.
- Pulls recently updated GitHub repositories related to Solana, Anchor, Jupiter, wallets, and agents.
- Scores narratives by keyword match strength, liquidity, volume, organic score, verification, repository freshness, stars, and breadth of evidence.
- Outputs 3-5 practical build ideas tied to the detected narratives.
- Publishes as a GitHub Pages-compatible static site.

## Data Sources

- Jupiter Tokens API V2 via `https://lite-api.jup.ag/tokens/v2`
- GitHub Search API via `https://api.github.com/search/repositories`

The dashboard uses `data/snapshot.json`, generated by `scripts/refresh.js`. The GitHub Action refreshes the snapshot every two weeks and can also be run manually.

## Run Locally

```bash
npm run refresh
npm run check
npm run serve
```

Then open `http://localhost:4173`.

## Scoring

Each signal is normalized before narrative scoring.

- Token score: source rank, 24h volume, liquidity, verification, organic score, and bucket breadth.
- Repository score: stars, forks, freshness, and number of matching search queries.
- Narrative score: weighted sum of matching token and repository signals plus breadth bonus.

The scoring is intentionally transparent and lightweight. It favors explainability over black-box prediction.

## Current Narrative Families
